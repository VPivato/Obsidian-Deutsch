Para relações de parentesco por afinidade, ou seja, que entram na família por casamento, há o prefixo „Schwieger“

| Wort             | Bedeutung |
| ---------------- | --------- |
| Schwiegervater   | Sogro     |
| Schwiegermutter  | Sogra     |
| Schwiegersohn    | Genro     |
| Schwiegertochter | Nora      |
E etc...

- - -

Para relações que não biológicas, que não compartilham sangue, há o prefixo „Stief“

| Wort        | Bedeutung  |
| ----------- | ---------- |
| Stiefmutter | Madrasta   |
| Stiefvater  | Padrasto   |
| Stiefbruder | Meio-irmão |
| Stiefsohn   | Enteado    |
 E etc...
