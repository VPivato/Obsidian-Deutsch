
No idioma alemão, se expressa negação por duas palavras: **kein** und **nicht**.
Cada uma dessas palavras possui um pequeno conjunto de regras sobre quando se deve usá-la.

# Kein

Tabela para revisão rápida sobre os **artigos de negação**[^1] em cada caso gramatical.

|                   | Maskulin | Feminin | Neutrum | Plural |
| ----------------- | -------- | ------- | ------- | ------ |
| **[[Nominativ]]** | kein     | keine   | kein    | keine  |
| **[[Akkusativ]]** | keinen   | keine   | kein    | keine  |
| **[[Dativ]]**     | keinem   | keiner  | keinem  | keinen |
| **[[Genitiv]]**   | keines   | keiner  | keines  | keiner |

**Erste Regel**: a primeira regra dita que deve-se usar _kein_ junto a substantivos com artigo indefinido[^2].
> Ist das ein Tisch?
> Nein, das ist _kein_ Tisch.

**Zweite Regel**: a segunda regra dita que deve-se usar _kein_ junto a substantivos sem artigo.
> Sind das Tomaten?
> Nein, das ist _keine_ Tomaten

**Dritte Regel**: a terceira regra dita que deve-se usar _kein_ junto a adjetivos atributivos[^3].
> Ist das ein neues Buch?
> Nein, das ist _kein_ neues Buch.

### Nicht

**Erste Regel**: deve-se usar _nicht_ junto a verbos.
> Ich verstehe das _nicht_.
> Sie geht _nicht_ das Hause.

**Zweite Regel**: deve-se usar _nicht_ junto a adjetivos.
> Das Sofa ist _nicht_ billig.
> Er wohnt _nicht_ allein.

**Dritte Regel**: deve-se usar _nicht_ junto a advérbios.
> Lernt du _nicht_ gern Deutsch?
> Wir essen morgen _nicht_ zu Hause.

**Vierte Regel**: deve-se usar _nicht_ junto a substantivos com artigo definido.
> Das ist _nicht_ die beste Pizza.
> Ich sehe das Problem _nicht_.

**Fünfte Regel**: deve-se usar _nicht_ junto a substantivos com artigo possessivo.
> Das ist _nicht_ mein Buch.
> Sind das nicht deine Stifte?
