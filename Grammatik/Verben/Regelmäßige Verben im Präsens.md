### Regelmäßige Verben
A conjugação de verbos regulares no tempo presente segue um padrão estrito, onde o **Verbstamm**[^1] permanece igual, e apenas o **Endung**[^1] muda:

|           | Singular           | Plural             |
| --------- | ------------------ | ------------------ |
| 1. Person | ich Verbstamm + e  | wir Verbstamm + en |
| 2. Person | du Verbstamm + st  | ihr Verbstamm + t  |
|           | Sie Verbstamm + en | Sie Verbstamm + en |
| 3. Person | er Verbstamm + t   | sie Verbstamm + en |
|           | sie Verbstamm + t  |                    |
|           | es Verbstamm + t   |                    |

Um exemplo com o verbo "Wohnen":

|           | Singular   | Plural     |
| --------- | ---------- | ---------- |
| 1. Person | ich wohne  | wir wohnen |
| 2. Person | du wohnst  | ihr wohnt  |
|           | Sie wohnen | Sie wohnen |
| 3. Person | er wohnt   | sie wohnen |
|           | sie wohnt  |            |
|           | es wohnt   |            |

> [!Caso especial - 1]
> Quando o **Verbstamm** termina em _-s_, _-z_ ou _-ß_:
> _-st_ se torna _-t_
> Wie heißst du? <span style="color: red">Falsch</span>
> Wie heißt du? <span style="color: green">Richtig</span>

> [!Caso especial - 2]
> Quando o **Verbstamm** termina em _-d ou _-t_:
> _-st_ se torna _-est_
> _-t_ se torna _-et_
> Wo arbeitst du? <span style="color: red">Falsch</span>
> Wo arbeitest du? <span style="color: green">Richtig</span>

> [!Caso especial - 3]
> Quando o **Verbstamm** termina em _-m_ ou _-n_ precedido por consoante:
> _-st_ se torna _-est_
> _-t_ se torna _-et_
> Wie atmst du? <span style="color: red">Falsch</span>
> Wie atmest du? <span style="color: green">Richtig</span>

Ambos os casos especiais 2 e 3 ocorrem porque a fonética alemã evita encontros consonantais impronunciáveis.
