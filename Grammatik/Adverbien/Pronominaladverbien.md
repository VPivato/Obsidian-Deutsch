#grammatik #adverbien

Para se referir a pessoas, o nome se torna „Pronominaladverb“.
Para mais informações sobre o uso com **coisas**, vide: [[Präpositionaladverbien]].

- - -
### Menschen
| Verb mit Präposition | Fragewort    | Pronominaladverbien |
| -------------------- | ------------ | ------------------- |
| träumen von          | Von wen...?  | Von ihm/ihr         |
| warten auf           | Auf wen...?  | Auf ihn/sie         |
| sprechen über        | Über wen...? | Über ihn/sie        |

A regra é „Präposition + Fragewort/Pronomen“.
Zum Beispiel:
- Von wem <small>(wer dem)</small> träumst du?
  _Com quem você sonha?_
- Ich träume von Brad Pitt.
  _Eu sonho com Brad Pitt._
- Von dem träume ich gar nicht.
  _Eu não sonho com ele de forma alguma._

Oder:
- Über wen sprecht ihr?
  _De quem vocês falam?_
- Wir sprechen über Peter.
  _Nós falamos sobre Peter._
- Über ihn spricht gerade de ganze Welt
  _O mundo inteiro está falando sobre ele._
