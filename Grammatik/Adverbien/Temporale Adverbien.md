#grammatik #adverbien

### Nie
Para expressar algo que nunca acontece.

Zum Beispiel:
- Wie oft spielst du Fußball?
  _Com que frequência você joga futebol?_
- Ich spiele nie Fußball.
  _Eu nunca jogo futebol _

### Fast nie
Para expressar algo que quase nunca acontece.

Zum Beispiel:
- Wie oft fährt Lisa Rad?
  _Com que frequência Lisa anda de bicicleta._
- Sie fährt fast nie Rad.
  _Ela quase nunca anda de bicicleta._

### Selten
Para expressar algo que é difícil acontecer.

Zum Beispiel:
- Wie oft spielt dein Vater Gitarre?
  _Com que frequência seu pai toca guitarra?_
- Er spielt selten Gitarre.
  _Ele raramente toca guitarra._

### Manchmal
Para expressar algo que às vezes acontece.

Zum Beispiel:
- Wie oft schaust du Netflix?
  _Com que frequência você assiste Netflix._
- Ich schaue manchmal Netflix.
  _Eu às vezes assisto Netflix._

### Oft
Para expressar algo que acontece com frequência.

Zum Beispiel:
- Wie oft kocht deine Mutter?
  _Com que frequência sua mãe cozinha?_
- Sie kocht oft.
  _Ela cozinha frequentemente._

### Immer
Para expressar algo que acontece sempre.

Zum Beispiel:
- Wie oft surft Tom im Internet?
  _Com que frequência Tom surfa na internet?_
- Er surft immer im Internet?
  _Ele sempre surfa na internet._

> [!WARNING] Uso de "immer"
> O uso desse advérbio se refere a algo que **sempre** acontece, dando a ideia de uma frequência 24/7. Para algo um pouco menos extremo, é mais comum usar "ständig".
> z.B.: Er surft ständig im Internet.

### Täglich / Jeden Tag
Para expressar algo que acontece diariamente.

Zum Beispiele:
- Wie oft machst du Sport?
  _Com que frequência você se exercita?_
- Ich mache täglich / jeden Tag Sport.
  _Eu faço esportes todo dia._

### Einmal, Zweimal ...
Para falar precisamente sobra a quantidade de vezes que algo acontece, usa-se os seguintes advérbios:
> Einmal, zweimal, dreimal, viermal, fünfmal ...

Zum Beispiel:
- Wie oft liest du?
  _Com que frequência você lê?_
- (Ich lese) Einmal pro Woche.
  _(Eu leio) Uma vez por mês._

Oder:
- Wie oft geht Martha shoppen?
  _Com que frequência Martha vai às_ compras?
- Zweimal im Monat.
  _Duas vezes ao mês_

### Wochentage und Tageszeiten
Também é possível transformar os [[Die Wochentage|dias da semana]] e os [[Die Tageszeiten|períodos do dia]] em advérbios, para isso, adiciona-se um "s" no final.
Eles expressam algo que acontece habitualmente nos horários especificados.

Zum Beispiele:
- Wann machst du Sport?
  _Quando você se exercita?_
- Ich mache morgens / mittags / abends Sport.
  _Eu me exercito de manhã / à tarde / ao anoitecer._
- Ich mache montags / mittwochs / freitags Sport.
  _Eu me exercito às segundas-feiras / quartas-feiras / sextas-feiras._
