#grammatik #adverbien 

São palavras usadas primariamente para **coisas** (Sachen), mas é possível usar para pessoas (Menschen). Têm a função de conectar frases e evitar repetições.
Normalmente usadas quando há um [[Verben mit Präpositionen]].

Para ver o uso em relação a pessoas, vide [[Pronominaladverbien]].

> [!TIP] das Fugen-R
> Se a preposição começa com uma vogal, adiciona um "R" para facilitar a pronúncia.
> z.B.: Woran, worauf, worüber.

> [!TIP] Reflexive Verben
> [[Reflexive Verben|Verbos Reflexivos]] frequentemente usam uma preposição, por isso é bem comum usar „wo/da + Präposition“.
> z.B.: sich freuen auf, sich interessieren für.

- - -
### Sachen
| Verb mit Präposition | Fragewort | Präpositionaladverb |
| -------------------- | --------- | ------------------- |
| träumen von          | wovon     | davon               |
| warten auf           | worauf    | darauf              |
| sprechen über        | worüber   | darüber             |

A regra geral é „wo/da + Präposition“. O „wo“ é para se fazer perguntas, e o „da“ é para se referir ao assunto já mencionado.
Zum Beispiel:
- Wovon träumst du?
  _Com o que você sonha?_
- Ich träume von einem Haus.
  _Eu sonho com uma casa._
- Davon träume ich auch.
  _Eu também sonho com isso._

Oder:
- Worauf wartest du?
  _Pelo que você espera?_
- Ich warte auf mein Resultat.
  _Eu espero pelo meu resultado._
- Darauf warte ich auch.
  _Eu também espero por isso._

Não é obrigatório que se tenha usado uma Fragewort para usar usar um Präpositionaladverb. No exemplo abaixo, o verbo é "rechnen mit etwas":
- Ich habe die Prüfung nicht bestanden. **Damit** habe ich nicht gerechnet.
  _Eu não passei na prova. Eu não esperava por **isso**._
