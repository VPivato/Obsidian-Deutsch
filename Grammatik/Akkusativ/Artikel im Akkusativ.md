
O caso acusativo muda **apenas** os artigos do gênero masculino, todos os outros se mantém iguais aos do [[Nominativ]]

|              | Def.  | Indef.  | Neg.     |
| ------------ | ----- | ------- | -------- |
| **Maskulin** | _den_ | _einen_ | _keinen_ |
| **Feminin**  | die   | eine    | keine    |
| **Neutrum**  | das   | ein     | kein     |
| **Plural**   | die   | -       | keine    |

- Ich sehe **den** Mann
  _Eu vejo o homem._
- - -
- Was macht die Frau?
  _O que a mulher está fazendo?_
- Sie schaut **einen** Film.
  _Ela está assistindo um filme._
- - -
- Was macht der Mann?
  _O que o homem está fazendo?_
- Er schreibt **einen** Brief.
  _Ele escreve uma carta_ <small>(carta é masculino em alemão)</small>
- - -
- Hast du **einen** Stift?
  _Você tem uma caneta?_
- Nein, ich habe **keinen** Stift.
  _Não, eu não tenho uma caneta._
