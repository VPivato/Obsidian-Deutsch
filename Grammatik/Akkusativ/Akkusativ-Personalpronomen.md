#grammatik #pronomen #akkusativ 

|               | Singular                   | Plural | Formalität |
| ------------- | -------------------------- | ------ | ---------- |
| **1. Person** | mich                       | uns    |            |
| **2. Person** | dich                       | euch   | informelle |
|               | Sie                        | Sie    | formelle   |
| **3. Person** | ihn <small>(masc.)</small> | sie    |            |
|               | sie <small>(fem.)</small>  |        |            |
|               | es <small>(neutro)</small> |        |            |

### 1. Person
- Siehst du mich?
  _Você me vê?_
- Rufst du uns an?
  _Você vai nos ligar?_

### 2. Person
- Ja, ich sehe dich.
  _Sim, eu te vejo._
- Ja, ich rufe euch an.
  _Sim, vou ligar para vocês._
- Ich sehe Sie.
  _Eu vejo o senhor / a senhora._
- Wir begrüßen Sie.
  _Nós cumprimentamos os senhores / as senhoras._

### 3. Person
- Magst du Hans? Nein, ich mag ihn nicht.
  _Você gosta do Hans? Não, eu não gosto dele._
- Hörst du Karina? Ja, ich höre sie.
  _Você ouve a Karina? Sim, eu a ouço._
- Findest du das Baby süß? Ja, ich finde es sehr süß.
  _Você acha esse bebê fofo? Sim, eu achei muito fofo._
- Kennst du die Leute? Nein, ich kenne sie nicht.
  _Você conhece essas pessoas? Não, eu não as conheço._
