#grammatik #artikel #nominativ

| Maskulin | Feminin | Neutrum | Plural |
| -------- | ------- | ------- | ------ |
| ein      | eine    | ein     | -      |

São artigos usados quando não está se falando de nenhuma pessoa ou objetivo específico, apenas **alguma** pessoa ou **algum** objeto.
O _Unbestimmte Artikel_ depende do gênero do **Substantiv**. Não existe artigo indefinido para o plural.

- Das ist **ein** Mann. *Der* Mann wohnt in Paris.
  _Este é um homem. O homem mora em Paris._
- Das ist **eine** Frau. *Die* Frau hört Musik.
  _Esta é uma mulher. A mulher escuta música._
- Das ist **ein** Mädchen. *Das* Mädchen lacht.
  _Esta é uma menina. A menina ri._
- Das sind Blumen. *Die* Blumen sind schön.
  _Estas são flores. As flores são lindas._
