#grammatik #pronomen #nominativ

|               | Singular                   | Plural | Formalität |
| ------------- | -------------------------- | ------ | ---------- |
| **1. Person** | ich                        | wir    |            |
| **2. Person** | du                         | ihr    | informelle |
|               | Sie                        | Sie    | formelle   |
| **3. Person** | er <small>(masc.)</small>  | sie    |            |
|               | sie <small>(fem.)</small>  |        |            |
|               | es <small>(neutro)</small> |        |            |

### 1. Person
- Ich wohne in Berlin.
  _Eu moro em Berlim_.
- Wir wohnen in Berlin.
  _Nós moramos em Berlim._

### 2. Person
- Wo wohnst du?
  _Onde você mora?_
- Wo wohnt ihr?
  _Onde vocês moram?_
- Herr Schulz, wo wohnen Sie?
  _Senhor Schulz, onde mora?_
- Herr und Frau Schulz, wo wohnen Sie?
  _Senhor e senhora Schulz, onde moram?_

### 3. Person
- Er wohnt in Berlin.
  _Ele mora em Berlim._
- Sie wohnt in Berlin.
  _Ela mora em Berlin._
- Das ist mein Buch, es ist alt.
  _Esse é o meu livro, ele é antigo._
- Das sind Susanne und Marcelo, sie wohnen in Berlin.
  _Estes são Susanne e Marcelo, eles moram em Berlin._
