#grammatik #artikel #possessivartikel #nominativ

|         | Maskulin | Feminin | Neutrum | Plural |
| ------- | -------- | ------- | ------- | ------ |
| **ich** | mein     | meine   | mein    | meine  |
| **du**  | dein     | deine   | dein    | deine  |
| **er**  | sein     | seine   | sein    | seine  |
| **sie** | ihr      | ihre    | ihr     | ihre   |
| **es**  | sein     | seine   | sein    | seine  |
| **wir** | unser    | unsere  | unser   | unsere |
| **ihr** | euer     | eure    | euer    | eure   |

- Ist das **dein** Stift?
  _Essa é sua caneta?_
- Ja, das ist **mein** Stift.
  _Sim, essa é minha caneta._
- - -
- Ist das **dein** Hund?
  _Esse é seu cachorro?_
- Nein, das ist **ihr** Hund?
  _Não, esse cachorro é dela._
- - -
- Ist das **sein** Vogel?
  _Esse pássaro é dele?_
- Ja, das ist sein Vogel.
  _Sim, esse pássaro é dele._
- - -
- Ist das **deine** Brille?
  _Esse óculos é seu?_
- Nein, das ist **seine** Brille.
  _Não, esses óculos são dele._
- - -
- Ist das **meine** Katze?
  _Esse é meu gato?_
- Nein, das ist **ihre** Katze.
  _Não, esse gato é dela._
- - -
- Sind das deine Handys?
  _Esses celulares são seus?_
- Ja, das sind meine Handys.
  _Sim, esses celulares são meus._
