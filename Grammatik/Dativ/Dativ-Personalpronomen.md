#grammatik #pronomen #dativ

|               | Singular                    | Plural | Formalität |
| ------------- | --------------------------- | ------ | ---------- |
| **1. Person** | mir                         | uns    |            |
| **2. Person** | dir                         | euch   | informelle |
|               | Ihnen                       | Ihnen  | formelle   |
| **3. Person** | ihm <small>(masc.)</small>  | ihnen  |            |
|               | ihr <small>(fem.)</small>   |        |            |
|               | ihm <small>(neutro)</small> |        |            |

### 1. Person
- Er hilft **mir**.
  _Ele me ajuda._
- Der Lehrer erklärt **uns** die Regel.
  _O professor explica a regra para nós._

### 2. Person
- Ich danke **dir**.
  _Eu te agradeço._
- Ich gebe **euch** Zeit
  _Eu dou tempo a vocês_
- Ich danke **Ihnen**.
  _Eu agradeço ao senhor / à senhora / aos senhores._

### 3. Person
- Ich gebe **ihm** das Buch.
  _Eu dou a ele o livro._
- Ich helfe **ihr**.
  _Eu ajudo ela._
- Das gehört **ihm**.
  _Isso pertence a ele/isso._
- Wir helfen **ihnen**
  _Nós ajudamos eles/elas._
