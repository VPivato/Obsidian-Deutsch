#grammatik #artikel #dativ 

| Maskulin | Feminin | Neutrum | Plural |
| -------- | ------- | ------- | ------ |
| einem    | einer   | einem   | -      |

- <span style="color:salmon">Ich</span> gebe <span style="color:lightgreen">einem Mann</span> <span style="color:lightblue">ein Buch</span>.
  _Eu dou um livro a um homem._
No exemplo acima:
_Ich_ — o sujeito — está no Nominativ.
_Einem Mann_ — o objeto indireto — está no Dativ.
_Ein Buch_ — o objeto direto — está no Akkusativ

- Er liest **einem** Mann den Text.
  _Ele lê o texto para um homem._
- Ich helfe **einer** Frau.
  _Eu ajudo uma mulher._
- Das Kind spielt mit **einem** Spielzug.
  _A criança brinca com um brinquedo._
