- <span style="color:salmon">Der Mann</span> gibt <span style="color:lightgreen">der Frau</span> <span style="color:lightblue">einen Brief</span>.
  _O homem dá uma carta à mulher._
No exemplo acima:
_Der Mann_ — o sujeito — está no Nominativ.
_Der Frau_ — o objeto indireto — está no Dativ.
_Einen Brief_ — o objeto direto — está no Akkusativ

- Der Kellner bringt **dem** Gast das Essen.
  _O garçom traz a comida para o convidado._
- Er schreibt **der** Mutter einen Brief.
  _Ele escreve uma carta para a mãe._
- Der Papa erzählt **dem** Kind eine Geschichte.
  _O pai conta uma história para a criança._
